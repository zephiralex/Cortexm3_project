Реализация обмена данными между потоками в системе FreeRTOS на базе микроконтроллера STM32F103C8T6.
Для взаимодействия потоков используются:
* очередь;
* мьютекс;
* бинарный семафор.

Вся разработка велась в среде Arduino IDE.

Использовалось следующее оборудование:
* Микроконтроллер STM32F103C8T6 (Ядро Cortex M3, 64кб ROM, 20кб RAM)
* Модуль USB to UART на контроллере CP2102;
* Программатор ST-Link v.2

Для решения задачи использовались различные обучающие материалы: 
* Освоение ядра реального времени FreeRTOSTM Ричарда Барри;
* выдержки из нейросетей для кратких примеров и объяснения некоторых функций;
* статьи на habr.ru
* видео ролики на YouTube
